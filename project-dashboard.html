Project Description

This project was developed as part of the Dashboard Visualization Competition at GELAR RASA 2024. The project focuses on analyzing Airbnb listing data to uncover pricing patterns, user preferences, and key factors influencing listing performance. My role in this project primarily involved data preprocessing, exploratory data analysis, and insight generation, where analytical findings were translated into dashboard requirements. Based on these insights, an interactive dashboard was designed and implemented collaboratively to support data storytelling and business-oriented decision making. The final output is an interactive Looker Studio dashboard that presents analytical results in a clear, structured, and actionable manner.

Price & Property Analysis

This section analyzes Airbnb pricing patterns across different districts, property types, and room types. Visualizations include average price comparisons, listing price distributions by district, and the impact of amenities count on pricing, helping identify pricing trends and market differences.

Ratings & Reviews Analysis

Provides insights into customer satisfaction by analyzing review scores across multiple categories such as cleanliness, location, communication, and accuracy. The dashboard also visualizes the relationship between listing prices and review ratings, as well as top-performing listings based on overall ratings.

Preference & Popularity Insights

Highlights user preferences and listing popularity based on the number of reviews. This feature allows comparison across districts and property types, helping identify which listings attract the most attention from users.

Process

Data Understanding & Problem Framing

The project began with understanding the competition case and identifying key business questions related to pricing, popularity, and user preferences within Airbnb listings.

Data Cleaning & Preparation

Raw datasets provided by the competition organizers were explored and cleaned to ensure consistency and reliability. This step included handling missing values, merging datasets, and preparing derived variables for analysis.

Exploratory Data Analysis

Exploratory analysis was conducted to identify relationships between listing attributes, pricing, ratings, and user behavior. Patterns and correlations discovered at this stage formed the basis for defining dashboard metrics and visualizations.

Insight Definition & Dashboard Planning

Based on analytical findings, key insights and storytelling goals were defined. These insights guided decisions on what metrics, charts, and interactions should be included in the final dashboard.

Dashboard Development & Validation

Insights were implemented into an interactive Looker Studio dashboard collaboratively. The dashboard was reviewed to ensure clarity, interpretability, and alignment with business decision-making needs.
